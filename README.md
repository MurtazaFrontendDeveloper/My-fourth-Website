# 🌐 Portfolio Website

A simple static portfolio-style website built using HTML and CSS. This project was created as part of my early learning in frontend development to practice basic layout and styling techniques.

---

## 📌 Overview

This website is a basic static project that focuses on practicing how to structure a webpage and apply styling using pure CSS. It is not meant to represent a full personal portfolio, but rather a learning project built to improve my frontend skills.

---

## 🛠️ Built With

* HTML5  
* CSS3  

---

## ✨ Features

* Simple and clean layout  
* Basic styling using CSS  
* Structured sections for a portfolio-style design  

---

## 🚧 Limitations

* Not responsive (optimized for desktop only)  
* No JavaScript or interactivity  
* Basic design (learning project)
